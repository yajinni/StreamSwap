Iframe Embedding & Customization Help

Because of web security settings (called X-Frame-Options), some websites do not allow themselves to be loaded within another site. Here is how to configure popular services:

YouTube

Use the format:
https://www.youtube.com/embed/[VIDEO_ID]

Example: https://www.youtube.com/embed/jfKfPfyJRdk

Twitch

Use the player format with your site's domain as a parent parameter:
https://player.twitch.tv/?channel=[CHANNEL]&parent=[YOUR_DOMAIN]

Example (local): https://player.twitch.tv/?channel=esl_csgo&parent=localhost

Vimeo

Use the player format:
https://player.vimeo.com/video/[VIDEO_ID]
